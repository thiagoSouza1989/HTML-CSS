Tags HTML

Em HTML podemos inserir os elementos a seguir. Passe o mouse para descobrir qual tag deverá utilizar em cada caso:

  • Parágrafo
    • Use a tag <p>
  • Título
    • Use a tag <h1> até <h6>
  • Imagens
    • Use a tag <img>
  • Abreviações
    • Use a tag <abbr>
  • Códigos-fontes
    • Use a tag <code>
  • Listas numeradas
    • Use a tag <ol> e <li>
  • Listas com demarcadores
    • Use a tag <ul> e <li>
